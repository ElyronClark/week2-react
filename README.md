# Week 2 React App

A full stack React app connected to an Express backend.

## Features
- Fetches skills list from Express API on load
- Add new skills via POST request to backend

## Setup

1. Install dependencies: `npm install`
2. Make sure the Express API is running on port 3001
3. Run: `npm run dev`
4. Open: `http://localhost:5173`

## Stack 
- React 18
- Vite
- Express (backend — see week2-express repo)
